Operatore NAND

L'operatore NAND è un operatore logico ( operatore booleano ) di negazione della congiunzione logiche. È anche detto AND negato. Date due proposizioni logiche A e B, l'operatore NAND determina una terza proposizione logica C che si manifesta falsa quandole proposizioni A e B sono entrambe vere e si manifesta vera in tutti gli altri casi. Il simbolo dell'operatore NAND è una freccia orientata verso l'alto.

A B A and B
True True False
True False True
False False True
False False True
